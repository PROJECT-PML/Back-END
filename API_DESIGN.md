## API规范
### 获取新闻列表
 GET /news/{type_id}/offset=1&limit=20
 ```json
 {
     "status": 200,
     "msg": "OK",
     "data": [
         {
        "Id": 1,
        "title":"赵丽颖现身机场，疲惫过度助理忙搀扶",
        "CreatedAt":"2018-12-08 20:21",
         },
          {
         "Id": 2,
         "title":"美国为何如此忌惮中国：美将军终于说了大实话！",
         "CreatedAt":"2018-12-08 20:21",
          }
     ]
    
 }
 ```

### 获取新闻评论
GET /news/news_id/comments
 ```json
 {
     "status": 200,
     "msg": "OK",
     "data": [
         {
        "Id": 1,
         "News_id": 1
         "Creator":2
        "CreatedAt":"2018-12-08 22:11",
        "Content": "hhhh,233333"
         },
          {
        "Id": 1,
         "News_id": 1
         "Creator":1
        "CreatedAt":"2018-12-08 22:11",
        "Content": "来试试"
         }  
     ]
    
 }
 ```


### 新闻添加评论
POST /news/{news_id}/comments/{comment_id}
 ```json
 {
     "status": 200,
     "msg": "OK",
     "data": 
         {
         "Id": 1,
         "News_id": 1
         "Creator":2
        "CreatedAt":"2018-12-08 22:11",
        "Content": "hhhh,233333"
         }
    
 }
 ```
### 用户添加新闻收藏
POST /news/{news_id}/comments/{comment_id}
 ```json
 {
     "status": 200,

        "CreatedAt":"2018-12-08 20:21",
        "Content": ""
         }
     ]
    
 }
 ```
